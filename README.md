BistroKASA - Pokladni system pro pizzerie a bistra
BistroKASA je profesionalni pokladni software urceny pro pizzerie, kebabarny a jina gastro zarizeni. Aplikace bezi jako samostatny spustitelny soubor (.exe) a nevyzaduje instalaci Pythonu ani jinych programovacich prostredku.

Rychle objednavky: Moduly pro pizzu, kebab a napoje.

Pulena pizza: Unikatni vizualni editor pro vytvoreni pizzy "dva v jednom" s automatickym prepoctem ceny.

Vlastni pizza: Moznost sestaveni pizzy presne podle prani zakaznika s vyberem zakladu a ingredienci.

Cloudova synchronizace: Moznost odesilani trzeb na vzdaleny webovy server pro online prehled majitele.

Bezpecnost a zalohovani: Automaticke zalohy databaze a system licencovani vazany na hardware pocitace (HWID).

Instalace a spusteni
Stahnete soubor BistroKASA.exe (nebo podobny nazev) z teto stranky.

Umistete soubor do libovolne slozky ve vasem pocitaci.

Spustte aplikaci dvojklikem na soubor .exe.

Pri prvnim spusteni system vytvori v domovske slozce uzivatele (C:/Users/Jmeno/Kasa) potrebne soubory pro databazi a konfiguraci.

Systemove pozadavky

BistroKASA - POS System for Pizzerias and Bistros
BistroKASA is a professional POS software designed for pizzerias, kebab shops, and other food service establishments. The application runs as a standalone executable (.exe) and does not require Python or any other programming environment to be installed.

Key Features
Fast Ordering: Dedicated modules for pizza, kebab, and drinks.

Half-Pizza Tool: A unique visual editor for creating "two-in-one" pizzas with automatic price calculation.

Custom Pizza: Build pizzas exactly to customer specifications with a choice of bases and toppings.

Receipt Printing: Optimized for 58mm thermal printers (Xprinter, EPSON, etc.). Automatically prints customer receipts and kitchen tickets.

Customer Directory: Contact management with automatic data population based on phone numbers.

Statistics and Reporting: Detailed sales overviews, item popularity charts, and password-protected daily closings.

Cloud Synchronization: Option to send sales data to a remote web server for the owner's online monitoring.

Security and Backups: Automatic database backups and a licensing system tied to the computer hardware (HWID).

Installation and Startup
Download the BistroKASA.exe file (or similar name) from this repository.

Place the file in any folder on your computer.

Launch the application by double-clicking the .exe file.

Upon the first run, the system will create necessary database and configuration files in the user's home directory (C:/Users/Name/Kasa).

System Requirements
Operating System: Windows 10 or Windows 11.

Hardware: Any PC or laptop with a USB port for the printer.

Printer (Optional): Any thermal receipt printer with Windows drivers.

Technical Notes
Data is stored locally in an SQLite database (~/Kasa/pizzeria.db).

Pricing and menu configuration can be done directly within the app (requires admin password).

The program may be flagged by Windows SmartScreen during the first launch (since it's an unsigned app; select "Run anyway").
